# Pet Grok v1.1.1

Release date: 2026-07-08

This patch makes Pet Grok substantially easier to download and safer to distribute.

## Highlights

- Downloadable macOS Apple silicon, macOS Intel, and Windows installers
- Automated GitHub Release packaging for future version tags
- Electron runtime updated to a patched version
- Sleepy pet renamed to the original **Doze Buddy** identity
- Durable download link and platform guidance added to the README

## Installation note

The installers are currently unsigned. macOS users may need to right-click the app and choose **Open** on first launch. Windows users may need to choose **More info → Run anyway** in SmartScreen.
